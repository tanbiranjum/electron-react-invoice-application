# Invoice Application

Invoice application to manage invoice and print. Made with Electron,React,Firebase,Tailwind.css

## Tools

- React
- Electron
- Tailwind.css
- Firebase
- React Router
- Redux
- Redux Toolkit

## Installation

Install this with pnpm

```bash
  pnpm install
```

To run this app

```bash
  pnpm electrondev
```
